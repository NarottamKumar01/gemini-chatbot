# 🤖 Gemini Chatbot

A simple chatbot built using **Python**, **Streamlit**, and the **Google Gemini API**. It accepts user queries and generates AI-powered responses.

## Features

- Ask questions using a simple interface
- AI-generated responses with Gemini API
- Easy to run locally

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

Create a `.env` file:

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```
